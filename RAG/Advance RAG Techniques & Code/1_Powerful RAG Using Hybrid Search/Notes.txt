#### RAG Architecture
1. [INGESTION]
Data → Chunking → Embedding → Database

2. [RETRIEVAL]
User Query → Embedding → Similarity Search → Database
            → Ranked Results → Context (Top K) → LLM → Final Answer

#### Different types of retrieval techniques
1. Naive retrieval + Keyword Search (Hybrid search)
2. Sentence window retrieval
3. Self Query retrieval
